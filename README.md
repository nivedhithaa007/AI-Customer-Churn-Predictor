AI Customer Churn Predictor

## Overview

AI Customer Churn Predictor is a Machine Learning application that predicts whether a telecom customer is likely to leave a service based on customer demographics, subscription details, and usage patterns.

The project uses the Telco Customer Churn Dataset and a Logistic Regression model to analyze customer information and predict churn behavior.

---

## Features

* Customer churn prediction using Machine Learning
* Interactive web interface built with Streamlit
* Real-world Telco Customer Churn dataset
* Customer information analysis
* Binary classification (Stay / Leave)
* User-friendly prediction dashboard

## Dataset

Dataset Used:
**Telco Customer Churn Dataset**

The dataset contains customer demographic information, account details, service subscriptions, and churn status.

Key attributes include:

* Gender
* Partner Status
* Dependents
* Tenure
* Phone Service
* Monthly Charges
* Total Charges
* Churn Status

## Installation

Clone the repository:

```bash
git clone https://github.com/nivedhithaa007/AI-Customer-Churn-Predictor.git
```

Navigate to the project directory:

```bash
cd AI-Customer-Churn-Predictor
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
